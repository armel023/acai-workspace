# ADR-006 - Use React, Vite, and Material UI

## Status

Accepted

## Date

2026-06-08

---

## Context

Acai Workspace requires:

- Rapid development
- Type safety
- Enterprise UI components
- Strong ecosystem support

---

## Decision

Use:

- React
- TypeScript
- Vite
- Material UI

---

## Consequences

### Positive

- Mature ecosystem
- Excellent TypeScript support
- Enterprise-grade components
- Fast development experience

### Negative

- Bundle size larger than minimal UI libraries

---

## Alternatives Considered

### Angular

Rejected due to increased complexity for the current team size.

### Next.js

Rejected because SSR is not currently required.

### Ant Design

Rejected in favor of Material UI ecosystem familiarity.
